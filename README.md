# Restaurant Page

A simple Single Page Application (SPA) restaurant website built with **JavaScript**, **Webpack**, and **DOM Manipulation** as part of **The Odin Project** curriculum.

## ✨ Features

* Dynamic page rendering using JavaScript
* Single Page Application (SPA) architecture
* Home, Menu, and Contact pages
* Tab navigation without page reloads
* Modular JavaScript using ES Modules
* Responsive and clean user interface
* Bundled with Webpack

## 🛠️ Built With

* JavaScript (ES6 Modules)

## ▶️ Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd restaurant-page
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```
